隱私權政策

英翻中翻譯器（English to Chinese Translator）

我們收集哪些資料？

本插件不收集、不儲存、不傳輸任何個人資料。

API Key

您輸入的 Claude API Key 僅儲存於您的裝置本機（chrome.storage.local），不會上傳至任何伺服器，也不會與第三方共享。

翻譯內容

您選取或請求翻譯的文字會直接從您的瀏覽器傳送至 Anthropic API（api.anthropic.com）進行處理。本插件開發者無法存取這些內容。Anthropic 的資料處理方式請參閱其隱私權政策。

Token 用量統計

累積的 Token 用量數字儲存於您的裝置本機，僅供您個人參考，不會傳送至任何外部服務。

聯絡方式

如有任何疑問，請透過 GitHub Issues 聯繫。
